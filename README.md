# Biomodule 蓝牙SDK 

## 简介

本 SDK 包含回车生物电采集模块的蓝牙连接和生物电采集控制。通过集成BLE基础SDK可以实现和我们的采集模块连接，并控制其进行数据的采集和停止等指令，
集成BLE UI SDK可以方便为你的app搭建设备管理界面。

## Demo
